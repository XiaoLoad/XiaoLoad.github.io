Java 学习笔记

1.数组

数组：是一种容器，可以存放同种数据类型的多个值。

  • 数组容器在存放数据的时候，需要结合隐式转换考虑。
  • 例如：int 类型的数组容器可以存放 byte short int
    • double 类型的可以存放 byte short int long float double
  • 元素应该与数组容器的类型保持一致。
